As with Theorem 3.3.4, Theorem 3.4.3 lets us conclude that the graph of a function is concave up (or down) on a closed interval, assuming that the function is continuous on that interval. Again, we follow the convention that when a problem asks us to give the intervals on which the graph is concave up or down, we give open intervals, even if a closed interval is technically correct.
If a function has the same concavity on adjacent intervals \((a,b)\) and \((b,c)\text{,}\) and the function is continuous at \(b\text{,}\) we should combine the intervals, and state the result as \((a,c)\text{.}\) However, if \(b\) is a point of discontinuity, we must omit it from our intervals.
