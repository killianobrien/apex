We start by making a table of \(\cos(2\theta)\) evaluated at common angles \(\theta\text{,}\) as shown in Figure 10.4.14. These points are then plotted in Figure 10.4.15.(a). This particular graph “moves” around quite a bit and one can easily forget which points should be connected to each other. To help us with this, we numbered each point in the table and on the graph.
Using more points (and the aid of technology) a smoother plot can be made as shown in Figure 10.4.15.(b). This plot is an example of a rose curve.
