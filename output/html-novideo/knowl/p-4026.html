  1. Let \(\dx\) represent a small, nonzero change in \(x\) value.
  2. Let \(dx\) represent a small, nonzero change in \(x\) value (i.e., \(\dx = dx\)).
  3. Let \(\dy\) be the change in \(y\) value as \(x\) changes by \(\dx\text{;}\) hence
    \begin{equation*} \dy = f(x+\dx)-f(x)\text{.} \end{equation*}
  4. Let \(dy = \fp(x)dx\) which, by (4.4.1), is an approximation of the change in \(y\)-value as \(x\) changes by \(\dx\text{;}\) \(dy \approx \dy\text{.}\)
