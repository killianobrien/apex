It is useful to sketch the function in the integrand, as shown in Figure 5.2.13. We see we need to compute the areas of two regions, which we have labeled \(R_1\) and \(R_2\text{.}\) Both are triangles, so the area computation is straightforward:
\begin{equation*} R_1: \frac12(4)(8) = 16 \qquad R_2: \frac12(3)6 = 9\text{.} \end{equation*}
Region \(R_1\) lies under the \(x\)-axis, hence it is counted as negative area (we can think of the triangle's height as being “\(-8\)”), so
\begin{equation*} \int_{-2}^5(2x-4)\, dx = -16+9 = -7\text{.} \end{equation*}
