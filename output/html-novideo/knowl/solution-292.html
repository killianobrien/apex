Solution 5.5.11.1.

We refer back to Table 5.5.5 for the table of values of \(\sin(x^3)\text{.}\) Recall that \(\dx = 3\pi/40 \approx 0.236\text{.}\) Thus we have:
\begin{align*} \amp \int_{-\frac{\pi}4}^{\frac{\pi}2} \sin(x^3)\, dx\\ \amp \approx \frac{0.236}{2}\Big[-0.4657 + 2\Big(-0.1654+(-0.031)+\ldots+ 0.68999\Big)+(-0.67)\Big]\\ \amp = 0.4258\text{.} \end{align*}
The actual answer, accurate to 4 decimal places is \(0.4609\text{.}\) So the Trapezoidal Rule with 10 subintervals is an under-approximation by about \(0.0351\text{.}\)
