We note that we could actually show that \(\lim_{x\to c} e^x = e^c\) for any constant \(c\text{.}\) We do this by factoring out \(e^c\) from both sides, leaving us to show \(\lim_{x\to c} e^{x-c} = 1\) instead. By using the substitution \(u=x-c\text{,}\) this reduces to showing \(\lim_{u\to 0} e^u = 1\) which we just did in the last example.
