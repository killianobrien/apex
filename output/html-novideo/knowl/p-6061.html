Because we are considering \(a>0\) and \(x=a\sin(\theta)\text{,}\) which means \(\theta = \sin^{-1}(x/a)\text{.}\) The arcsine function has a domain of \(-\pi/2\leq \theta \leq \pi/2\text{;}\) on this domain, \(\cos(\theta) \geq 0\text{,}\) so \(a\cos(\theta)\) is always non-negative, allowing us to drop the absolute value signs.
