Solution 15.4.16.1.

By referencing Figure 15.4.17, we see we can make a closed path \(C\) that combines \(C_1\) with \(C_2\text{,}\) where \(C_2\) is traversed with its opposite orientation. We label the enclosed region \(R\text{.}\) Since \(\divv \vec F = 0\text{,}\) the Divergence Theorem states that
\begin{equation*} \oint_C \vec F\cdot \vec n\, ds = \iint_R \divv \vec F\, dA = \iint_R 0\, dA = 0\text{.} \end{equation*}
Using the properties and notation given in Theorem 15.3.7, consider:
\begin{align*} 0 \amp = \oint_C \vec F\cdot \vec n\, ds\\ \amp = \int_{C_1} \vec F\cdot \vec n\, ds +\int_{C_2^*} \vec F\cdot \vec n\, ds\\ \end{align*}

(where \(C_2^*\) is the path \(C_2\) traversed with opposite orientation)

\begin{align*} \amp = \int_{C_1} \vec F\cdot \vec n\, ds -\int_{C_2} \vec F\cdot \vec n\, ds.\\ \int_{C_2} \vec F\cdot \vec n\, ds\amp = \int_{C_1} \vec F\cdot \vec n\, ds\text{.} \end{align*}
Thus the flux across each path is equal.
