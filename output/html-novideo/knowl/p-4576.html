An object is thrown straight up with a velocity, in ft/s, given by \(v(t) = -32t+64\text{,}\) where \(t\) is in seconds, from a height of \(48\) feet.
  1. What is the object’s maximum velocity?
  2. What is the object’s maximum displacement?
  3. When does the maximum displacement occur?
  4. When will the object reach a height of \(0\text{?}\) (Hint: find when the displacement is \(-48\)ft.)
