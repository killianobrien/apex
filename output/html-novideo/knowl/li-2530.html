Item 2.
\begin{align*} f(x,y) \amp = \frac{x^3}{y^2} = x^3y^{-2}\\ f_x(x,y) \amp = \frac{3x^2}{y^2}\\ f_y(x,y) \amp = -\frac{2x^3}{y^3}\\ f_{xx}(x,y) \amp = \frac{\partial}{\px}\big(f_x\big) = \frac{\partial}{\px}\big(\frac{3x^2}{y^2}\big)\\ \amp = \frac{6x}{y^2}\\ f_{yy}(x,y) \amp = \frac{\partial}{\py}\big(f_y\big) = \frac{\partial}{\py}\big(-\frac{2x^3}{y^3}\big)\\ \amp = \frac{6x^3}{y^4}\\ f_{xy}(x,y) \amp = \frac{\partial}{\py}\big(f_x\big) = \frac{\partial}{\py}\big(\frac{3x^2}{y^2}\big)\\ \amp = -\frac{6x^2}{y^3}\\ f_{yx}(x,y) \amp = \frac{\partial}{\px}\big(f_x\big) = \frac{\partial}{\px}\big(-\frac{2x^3}{y^3}\big)\\ \amp = -\frac{6x^2}{y^3} \end{align*}
