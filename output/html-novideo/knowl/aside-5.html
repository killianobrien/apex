In this text, when we use the term “closed interval”, we mean an interval of the form \([a,b]\text{,}\) where \(a\) and \(b\) are real numbers. One may be surprised to learn that intervals of the form \([a,\infty)\text{,}\) \((-\infty, b]\) and even \((-\infty,\infty)\) are all also considered closed in advanced calculus. While the mathematics supported by this definition of closed is fascinating and important, it is beyond the scope of our purposes here.
Some results, such as The Extreme Value Theorem, are valid for intervals of the form \([a,b]\text{,}\) but not for intervals such as \([a,\infty)\text{.}\) The latter interval is closed, but not bounded.
A set of real numbers is bounded if there is a number that is greater than every element in the set (an upper bound), and a number that is less than every element in the set (a lower bound). When we do calculus in higher dimensions, we can no longer talk about intervals, but we can still talk about sets being closed and bounded. See Section 13.8 for details.
