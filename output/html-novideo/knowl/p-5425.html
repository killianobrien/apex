This can be integrated using Substitution. Set \(u = x/5\text{,}\) hence \(du = dx/5\) or \(dx=5\, du\text{.}\) Thus
\begin{align*} \int\frac{1}{25+x^2}\, dx \amp = \frac{1}{25}\int \frac{1}{1+\left(\frac{x}{5}\right)^2}\, dx\\ \amp = \frac15\int \frac{1}{1+u^2}\, du\\ \amp = \frac15\tan^{-1}(u) + C\\ \amp = \frac15\tan^{-1}\left(\frac x5\right)+C \end{align*}
