Exercise 18.
An object is thrown straight up with a velocity, in ft/s, given by \(v(t) = -32t+96\text{,}\) where \(t\) is in seconds, from a height of \(64\) feet.
  1. What is the object’s initial velocity?
  2. When is the object’s displacement 0?
  3. How long does it take for the object to return to its initial height?
  4. What is the maximum height the object reaches?
