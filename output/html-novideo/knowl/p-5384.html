Choosing \(u = 1/x\) makes \(du = -1/x^2\, dx\text{;}\) that does not seem helpful. However, setting \(u = \ln(x)\) makes \(du = 1/x\, dx\text{,}\) which is part of the integrand. Thus:
\begin{align*} \int \frac1{x\ln(x)}\, dx \amp = \int \frac{1}{\underbrace{\ln(x)}_{u}}\underbrace{\frac1x\, dx}_{du}\\ \amp = \int \frac1u\, du\\ \amp = \ln\abs{u} + C\\ \amp = \ln\abs{ \ln(x) } + C\text{.} \end{align*}
