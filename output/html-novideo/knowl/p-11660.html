The first two equations of the line are useful when a \(t\) value is given: one can immediately find the corresponding point on the line. These forms are good when calculating with a computer; most software programs easily handle equations in these formats. (For instance, the graphics program that made Figure 11.5.5 can be given the input “(2-t,3+t,1+2*t)” for \(-1\leq t\leq 3\text{.}\)).
