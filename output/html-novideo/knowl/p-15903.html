\begin{align*} 0 \amp = \oint_C \vec F\cdot \vec n\, ds\\ \amp = \int_{C_1} \vec F\cdot \vec n\, ds +\int_{C_2^*} \vec F\cdot \vec n\, ds\\ \end{align*}

(where \(C_2^*\) is the path \(C_2\) traversed with opposite orientation)

\begin{align*} \amp = \int_{C_1} \vec F\cdot \vec n\, ds -\int_{C_2} \vec F\cdot \vec n\, ds.\\ \int_{C_2} \vec F\cdot \vec n\, ds\amp = \int_{C_1} \vec F\cdot \vec n\, ds\text{.} \end{align*}
Thus the flux across each path is equal.
