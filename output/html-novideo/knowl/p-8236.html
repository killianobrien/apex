  1. A sequence \(\{a_n\}\) is monotonically increasing if \(a_n \leq a_{n+1}\) for all \(n\text{,}\) i.e.,
    \begin{equation*} a_1 \leq a_2 \leq a_3 \leq \cdots a_n \leq a_{n+1} \cdots \end{equation*}
  2. A sequence \(\{a_n\}\) is monotonically decreasing if \(a_n \geq a_{n+1}\) for all \(n\text{,}\) i.e.,
    \begin{equation*} a_1 \geq a_2 \geq a_3 \geq \cdots a_n \geq a_{n+1} \cdots \end{equation*}
  3. A sequence is monotonic if it is monotonically increasing or monotonically decreasing.
