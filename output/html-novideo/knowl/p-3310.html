The second derivative test can only be used on a function that is twice differentiable at \(c\text{.}\) For functions that are not twice differentiable at \(c\text{,}\) you will need to use the First Derivative Test. If you've already determined the sign diagram for \(\fp\text{,}\) the First Derivative Test is usually easier to apply, and it applies in cases when First Derivative Test does not.
