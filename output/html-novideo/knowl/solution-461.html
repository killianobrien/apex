Solution 7.5.3.1.

From Example 7.5.2 we know the total work performed is \(1,164.24\) J. We want to find a height \(h\) such that the work in pulling the rope from a height of \(x=0\) to a height of \(x=h\) is \(582.12\text{,}\) or half the total work. Thus we want to solve the equation
\begin{equation*} \int_0^h 0.6468(60-x)\, dx = 582.12 \end{equation*}
for \(h\text{.}\)
\begin{align*} \int_0^h 0.6468(60-x)\, dx \amp = 582.12\\ \left(38.808x-0.3234x^2\right)\Big|_0^h \amp =582.12\\ 38.808h-0.3234h^2 \amp =582.12\\ -0.3234h^2+38.808h-582.12 \amp =0 .\\ \end{align*}

Apply the Quadratic Formula:

\begin{align*} h\amp =17.57 \,\text{ and } \,102.43 \end{align*}
As the rope is only 60 m long, the only sensible answer is \(h=17.57\text{.}\) Thus about half the work is done pulling up the first 17.57 m; the other half of the work is done pulling up the remaining 42.43 m.
