  1. If \(\lim\limits_{n\to\infty}\dfrac{a_{n+1}}{a_n}\lt 1\text{,}\) then \(\ds\infser a_n\) converges.
  2. If \(\lim\limits_{n\to\infty}\dfrac{a_{n+1}}{a_n} \gt 1\) or \(\lim\limits_{n\to\infty}\frac{a_{n+1}}{a_n}=\infty\text{,}\) then \(\ds\infser a_n\) diverges.
  3. If \(\lim\limits_{n\to\infty}\dfrac{a_{n+1}}{a_n}=1\text{,}\) the Ratio Test is inconclusive.
