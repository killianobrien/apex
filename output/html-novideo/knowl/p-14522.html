Thus we have the following “curve to curve, point to point” bounds:
\begin{equation*} y^2/4\leq x\leq 2\sqrt y, \text{ and } 0\leq y\leq 4\text{.} \end{equation*}
Therefore,
\begin{align*} \iint_R (4-y)\, dA \amp = \int_0^4\int_{y^2/4}^{2\sqrt{y}}(4-y)\, dx\, dy\\ \amp = \int_0^4 \big(x(4-y)\big)\Big|_{y^2/4}^{2\sqrt{y}} dy\\ \amp = \int_0^4 \Big(\big(2\sqrt{y}-\frac{y^2}{4}\big)\big(4-y)\Big)\, dy\\ \amp = \int_0^4 \Big( \frac{y^3}{4}-y^2-2y^{3/2}+8y^{1/2}\Big)\, dy\\ \amp = \left.\left(\frac{y^4}{16}-\frac{y^3}{3}-\frac{4y^{5/2}}5+\frac{16y^{3/2}}3\right)\right|_0^4\\ \amp = \frac{176}{15} = 11.7\overline{3}\text{.} \end{align*}
