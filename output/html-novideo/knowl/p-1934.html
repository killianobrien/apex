Theorem 2.3.4 allows us to find the derivatives of a wide variety of functions. It can be used in conjunction with the Power Rule to find the derivatives of any polynomial. Recall in Example 2.1.17 that we found, using the limit definition, the derivative of \(f(x) = 3x^2+5x-7\text{.}\) We can now find its derivative without expressly using limits:
\begin{align*} \lzoo{x}{3x^2+5x-7} \amp = 3\lzoo{x}{x^2} + 5\lzoo{x}{x} - \lzoo{x}{7}\\ \amp = 3\cdot 2x+5\cdot 1- 0\\ \amp = 6x+5\text{.} \end{align*}
