Solution 1.2.14.1.
Let \(\varepsilon \gt 0\) be given. We wish to find \(\delta \gt 0\) such that when \(\abs{x-0}\lt\delta\text{,}\) \(\abs{f(x)-0}\lt\varepsilon\text{.}\)
First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-0}\lt\varepsilon\text{:}\)
\begin{equation*} \begin{aligned} \abs{\sin(x)-0}\amp\lt\varepsilon\\ \abs{\sin(x)}\amp\lt\varepsilon \end{aligned} \end{equation*}
Using the hint that \(\abs{\sin(x)} \leq \abs{x}\text{,}\) then if \(\abs{x}\lt0\text{,}\) it is automatic that \(\abs{\sin(x)}\lt0\text{.}\) This suggests we take \(\delta=\varepsilon\text{.}\)
Now we can apply the definition.
\begin{equation*} \begin{aligned} \abs{x-0}\amp\lt\delta\\ \abs{x}\amp\lt\varepsilon\\ \abs{\sin(x)}\amp\lt\varepsilon\\ \abs{\sin(x)-0}\amp\lt\varepsilon \end{aligned} \end{equation*}
In other words, \(\abs{x-0}\lt\delta\) implies \(\abs{\sin(x)-0}\lt\varepsilon\text{.}\) This is what we needed to prove.
