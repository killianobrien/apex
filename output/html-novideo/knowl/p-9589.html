But of course, there was a better way all along:
\begin{equation*} \ln(\sqrt{x}) = \ln(x^{1/2}) = \frac12\ln(x) \end{equation*}
using properties of the logarithm, and indeed, the result above is the same as the one we would have obtained by simply multiplying the Taylor series for \(\ln(x)\) by \(\frac12\text{.}\) Power series manipulation is a powerful technique, but one should not apply it blindly.
