Solution 1.2.8.1.
Let \(\varepsilon \gt 0\) be given. We wish to find \(\delta \gt 0\) such that when \(\abs{x-4}\lt\delta\text{,}\) \(\abs{f(x)-15}\lt\varepsilon\text{.}\)
First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-15}\lt\varepsilon\text{:}\)
\begin{equation*} \begin{aligned} \abs{\left(x^2+x-5\right)-15}\amp\lt\varepsilon\\ \abs{x^2+x-20}\amp\lt\varepsilon\\ \abs{x-4}\cdot\abs{x+5}\amp\lt\varepsilon\\ \abs{x-4}\amp\lt\frac{\varepsilon}{\abs{x+5}} \end{aligned} \end{equation*}
Since \(x\) is near \(4\text{,}\) we can safely assume that, for instance, \(3\lt x\lt 5\text{.}\) Thus
\begin{equation*} \begin{aligned} 3+5\amp\lt x+5\lt5+5\\ 8\amp\lt x+5\lt 10\\ \frac{1}{10}\amp\lt\frac{1}{x+5}\lt\frac{1}{8}\\ \frac{\varepsilon}{10}\amp\lt\frac{\varepsilon}{x+5}\lt\frac{\varepsilon}{8} \end{aligned} \end{equation*}
Since we need to begin the actual proof with \(\abs{x-4}\lt\delta\text{,}\) this suggests that we take \(\delta=\frac{\varepsilon}{10}\text{.}\)
Now we can apply the definition.
\begin{equation*} \begin{aligned} \abs{x-4}\amp\lt\delta\\ \abs{x-4}\amp\lt\frac{\varepsilon}{10}\\ \abs{x-4}\amp\lt\frac{\varepsilon}{\abs{x+5}}\\ \abs{x-4}\cdot\abs{x+5}\amp\lt\varepsilon\\ \abs{x^2+x-20}\amp\lt\varepsilon\\ \abs{\left(x^2+x-5\right)-15}\amp\lt\varepsilon \end{aligned} \end{equation*}
In other words, \(\abs{x-4}\lt\delta\) implies \(\abs{\left(x^2+x-5\right)-15}\lt\varepsilon\text{.}\) This is what we needed to prove.
