Now compare this with the density function \(\delta(x,y) = 2\sqrt{x^2+y^2}+1\text{.}\) Converting this to polar coordinates gives
\begin{equation*} \delta(r,\theta) = 2\sqrt{(r\cos(\theta) )^2+(r\sin(\theta) )^2}+1 = 2r+1\text{.} \end{equation*}
Thus the weight \(W\) is:
\begin{align*} W \amp = \int_0^{2\pi}\int_0^2 (2r+1)r\, dr\, d\theta\\ \amp = \int_0^{2\pi} (\frac23r^3+\frac12r^2)\Big|_0^2d\theta\\ \amp = \int_0^{2\pi} \left(\frac{22}3\right)\, d\theta\\ \amp = \frac{44}3\pi \approx 46.08\text{ lb } \text{.} \end{align*}
