Constant Rule
\(\lzoo{x}{c} = 0\text{,}\) where \(c\) is a constant.
Power Rule
\(\lzoo{x}{x^n}= nx^{n-1}\text{,}\) where \(n\) is an integer, \(n \gt 0\text{.}\)
Other common functions
\(\lzoo{x}{\sin(x)} = \cos(x)\)
\(\lzoo{x}{\cos(x)} = {-\sin(x)}\)
\(\lzoo{x}{e^x} = e^x\)
\(\lzoo{x}{\ln(x)} = \frac{1}{x}\text{,}\) for \(x \gt 0\text{.}\)
