This domain of this function was found in Example 13.1.3 to be \(D = \{(x,y)\,|\,\frac{x^2}9+\frac{y^2}4\leq 1\}\text{,}\) the region bounded by the ellipse \(\frac{x^2}9+\frac{y^2}4=1\text{.}\) Since the region includes the boundary (indicated by the use of “\(\leq\)”), the set contains all of its boundary points and hence is closed. The region is bounded as a disk of radius 4, centered at the origin, contains \(D\text{.}\)
