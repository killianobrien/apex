Theorem 9.4.4. Root Test.

Let \(\{a_n\}\) be a positive sequence, and consider \(\lim\limits_{n\to \infty} (a_n)^{1/n}\text{.}\)
  1. If \(\lim\limits_{n\to \infty} (a_n)^{1/n}\lt 1\text{,}\) then \(\ds\infser a_n\) converges.
  2. If \(\lim\limits_{n\to \infty} (a_n)^{1/n} \gt1\) or \(\lim\limits_{n\to \infty} (a_n)^{1/n}=\infty\text{,}\) then \(\ds\infser a_n\) diverges.
  3. If \(\lim\limits_{n\to \infty} (a_n)^{1/n}=1\text{,}\) the Root Test is inconclusive.
