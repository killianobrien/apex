Solution 14.1.12.1.

Once again we'll find the area of the region using both orders of integration.
Using \(dy\, dx\text{:}\)
\begin{equation*} \int_0^2\int_{x^2}^{2x}1\, dy \, dx = \int_0^2(2x-x^2)\, dx = \big(x^2-\frac13x^3\big)\Big|_0^2 = \frac43\text{.} \end{equation*}
Using \(dx\, dy\text{:}\)
\begin{equation*} \int_0^4\int_{y/2}^{\sqrt{y}} 1\, dx\, dy = \int_0^4 (\sqrt{y}-y/2)\, dy = \left(\frac23y^{3/2} - \frac14y^2\right)\Big|_0^4 = \frac43\text{.} \end{equation*}
