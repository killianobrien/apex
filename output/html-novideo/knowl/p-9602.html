We include the “unsimplified” expressions for the coefficients found in Example 9.6.10 as we are looking for a pattern. It can be shown that \(a_n = 2^n/n!\text{.}\) Thus the solution, written as a power series, is
\begin{equation*} y = \infser[0] \frac{2^n}{n!}x^n = \infser[0] \frac{(2x)^n}{n!}\text{.} \end{equation*}
