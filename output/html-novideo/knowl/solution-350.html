Solution 6.5.6.1.

Key Idea 6.5.1 presumes that the degree of the numerator is less than the degree of the denominator. Since this is not the case here, we begin by using polynomial division to reduce the degree of the numerator. We omit the steps, but encourage the reader to verify that
\begin{equation*} \frac{x^3}{(x-5)(x+3)} = x+2+\frac{19x+30}{(x-5)(x+3)}\text{.} \end{equation*}
Using Key Idea 6.5.1, we can rewrite the new rational function as:
\begin{equation*} \frac{19x+30}{(x-5)(x+3)} = \frac{A}{x-5} + \frac{B}{x+3} \end{equation*}
for appropriate values of \(A\) and \(B\text{.}\) Clearing denominators, we have
The values of \(A\) and \(B\) can be quickly found using the technique described in Example 6.5.5, or they can be found by equating coefficients, as we do in Example 6.5.6.
\begin{align*} 19x+30 \amp = A(x+3) + B(x-5)\\ \amp = (A+B)x + (3A-5B).\\ \end{align*}

This implies that:

\begin{align*} 19\amp = A+B\\ 30\amp = 3A-5B.\\ \end{align*}

Solving this system of linear equations gives

\begin{align*} 125/8 \amp =A\\ 27/8 \amp =B\text{.} \end{align*}
We can now integrate.
\begin{align*} \int \frac{x^3}{(x-5)(x+3)}\, dx \amp = \int\left(x+2+\frac{125/8}{x-5}+\frac{27/8}{x+3}\right)\, dx\\ \amp = \frac{x^2}2 + 2x + \frac{125}{8}\ln\abs{x-5} + \frac{27}8\ln\abs{x+3} + C\text{.} \end{align*}
