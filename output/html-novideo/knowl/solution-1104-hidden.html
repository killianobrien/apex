In Example 12.4.2, we found
\begin{equation*} \unittangent(t) = \la (-3/5)\sin(t) ,(3/5)\cos(t) ,4/5\ra\text{.} \end{equation*}
Therefore
\begin{equation*} \unittangentprime(t) = \la -\frac35\cos(t) ,-\frac35\sin(t) ,0\ra \text{ and } \norm{\unittangentprime(t)} = \frac35\text{.} \end{equation*}
Thus
\begin{equation*} \unitnormal(t) = \frac{\unittangentprime(t)}{3/5} = \la -\cos(t) ,-\sin(t) ,0\ra\text{.} \end{equation*}
We compute \(\unittangent(\pi/2) = \la -3/5,0,4/5\ra\) and \(\unitnormal(\pi/2) = \la 0,-1,0\ra\text{.}\) These are sketched in Figure 12.4.9.
