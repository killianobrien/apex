We find that
\begin{equation*} \frac{dz}{dt} = -\pi\sin(\pi t)\sin(2\pi t+\pi/2)+2\pi\cos(\pi t)\cos(2\pi t+\pi/2) \text{.} \end{equation*}
One can “easily” see that when \(t\) is an integer, \(\sin(\pi t) =0\) and \(\cos(2\pi t+\pi/2)=0\text{,}\) hence \(\frac{dz}{dt}=0\) when \(t\) is an integer. There are other places where \(z\) has a relative max/min that require more work. First, verify that \(\sin(2\pi t+\pi/2) = \cos(2\pi t)\text{,}\) and \(\cos(2\pi t+\pi/2) = -\sin(2\pi t)\text{.}\) This lets us rewrite \(\frac{dz}{dt} = 0\) as
\begin{equation*} -\sin(\pi t)\cos(2\pi t)-2\cos(\pi t)\sin(2\pi t)=0 \text{.} \end{equation*}
By bringing one term to the other side of the equality then dividing, we can get
\begin{equation*} 2\tan(2\pi t) = -\tan(\pi t) \text{.} \end{equation*}
Using the angle sum/difference formula for tangent, we know
\begin{equation*} \tan(2\pi t) = \tan(\pi t + \pi t) = \frac{\tan(\pi t)+\tan(\pi t)}{1-\tan^2(\pi t)} \text{.} \end{equation*}
Thus we write
\begin{equation*} 2\frac{\tan(\pi t)+\tan(\pi t)}{1-\tan^2(\pi t)} = -\tan(\pi t) \text{.} \end{equation*}
Solving for \(\tan^2(\pi t)\text{,}\) we find
\begin{equation*} \tan^2(\pi t) = 5 \implies \tan(\pi t) = \pm\sqrt{5} \text{,} \end{equation*}
and so
\begin{equation*} \pi t = \tan^{-1}(\pm\sqrt{5}) = \pm\tan^{-1}(\sqrt{5}) \text{.} \end{equation*}
Since the period of tangent is \(\pi\text{,}\) we can adjust our answer to be
\begin{equation*} \pi t = \pm\tan^{-1}(\sqrt{5})+ n\pi,\text{ where \(n\) is an integer. } \end{equation*}
Dividing by \(\pi\text{,}\) we find
\begin{equation*} t = \pm\frac1\pi\tan^{-1}(\sqrt{5})+ n,\text{ where \(n\) is an integer. } \end{equation*}
So the solutions in \([0,2)\) are \(0\text{,}\) \(\frac{1}{pi}\arctan(sqrt{5})\text{,}\) \(1-\frac{1}{pi}\arctan(sqrt{5})\text{,}\) \(1\text{,}\) \(1+\frac{1}{pi}\arctan(sqrt{5})\text{,}\) and \(2-\frac{1}{pi}\arctan(sqrt{5})\text{.}\)
