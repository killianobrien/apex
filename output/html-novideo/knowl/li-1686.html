Item 2.
The sequence \(\{b_n\}\) starts with 1, \(-1\text{,}\) 1, \(-1\text{,}\) \(\ldots\text{.}\) Consider some of the partial sums \(S_n\) of \(\{b_n\}\text{:}\)
\begin{align*} S_1 \amp = 1\\ S_2 \amp = 0\\ S_3 \amp = 1\\ S_4 \amp = 0 \end{align*}
This pattern repeats; we find that \(S_n = \begin{cases} 1 \amp n\, \text{ is odd } \\, 0 \amp n\, \text{ is even } \end{cases}\text{.}\) As \(\{S_n\}\) oscillates, repeating 1, 0, 1, 0, \(\ldots\text{,}\) we conclude that \(\lim\limits_{n\to\infty}S_n\) does not exist, hence \(\ds\infser (-1)^{n+1}\) diverges. A scatter plot of the sequence \(\{b_n\}\) and the partial sums \(\{S_n\}\) is given in Figure 9.2.3.(b). When \(n\) is odd, \(b_n = S_n\) so the marks for \(b_n\) are drawn oversized to show they coincide.
