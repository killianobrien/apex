  1. If \(\lim\limits_{n\to\infty} \dfrac{a_n}{b_n} = L\text{,}\) where \(L\) is a positive real number, then \(\ds \infser a_n\) and \(\ds \infser b_n\) either both converge or both diverge.
  2. If \(\lim\limits_{n\to\infty} \dfrac{a_n}{b_n} = 0\text{,}\) then if \(\ds \infser b_n\) converges, then so does \(\ds \infser a_n\text{.}\)
  3. If \(\lim\limits_{n\to\infty} \dfrac{a_n}{b_n} = \infty\text{,}\) then if \(\ds \infser b_n\) diverges, then so does \(\ds \infser a_n\text{.}\)
