Using the order \(dy\, dx\text{:}\)
\begin{align*} \iint_R\big(xy+e^y\big) \, dA \amp = \int_3^4\int_1^2\big(xy+e^y\big)\, dy \, dx\\ \amp = \int_3^4 \left(\left.\left[\frac12xy^2+e^y\right]\right|_1^2\, \right) dx\\ \amp = \int_3^4\left(\frac 32x + e^2-e\right)dx\\ \amp = \left.\left(\frac 34x^2 + \big(e^2-e\big)x\right)\right|_3^4\\ \amp = \frac {21}4+ e^2-e\approx 9.92\text{.} \end{align*}
