Theorem 2.7.17. Glossary of Derivatives of Elementary Functions.

Let \(f\) and \(g\) be differentiable functions, and let \(a\text{,}\) \(c\) and \(n\) be real numbers, \(a \gt 0\text{,}\) \(n\neq 0\text{.}\)
  1. \(\displaystyle \lzoo{x}{c} = 0\)
  2. \(\displaystyle \lzoo{x}{x} = 1\)
  3. \(\displaystyle \lzoo{x}{x^n} = nx^{n-1}\)
  4. \(\displaystyle \lzoo{x}{f(x)\pm g(x)} = f'(x) \pm g'(x)\)
  5. \(\displaystyle \lzoo{x}{c\cdot f(x)} = c\cdot f'(x)\)
  6. \(\displaystyle \lzoo{x}{f(x)\cdot g(x)} = f'(x)\cdot g(x)+f(x)\cdot g'(x)\)
  7. \(\displaystyle \lzoo{x}{f(g(x))} = f'(g(x)) \cdot g'(x)\)
  8. \(\displaystyle \lzoo{x}{\frac{f(x)}{g(x)}} = \frac{f'(x)\cdot g(x)-f(x) \cdot g'(x)}{(g(x))^2}\)
  9. \(\displaystyle \lzoo{x}{e^x} = e^x\)
  10. \(\displaystyle \lzoo{x}{\ln(x)} = \frac{1}{x}\)
  11. \(\displaystyle \lzoo{x}{a^x} = \ln(a)\cdot a^x\)
  12. \(\displaystyle \lzoo{x}{\log_a x} = \frac{1}{\ln(a)}\cdot\frac{1}{x}\)
  13. \(\displaystyle \lzoo{x}{\sin(x)} = \cos(x)\)
  14. \(\displaystyle \lzoo{x}{\cos(x)} = -\sin(x)\)
  15. \(\displaystyle \lzoo{x}{\tan(x)} = \sec^2(x)\)
  16. \(\displaystyle \lzoo{x}{\csc(x)} = -\csc(x)\cot(x)\)
  17. \(\displaystyle \lzoo{x}{\sec(x)} = \sec(x)\tan(x)\)
  18. \(\displaystyle \lzoo{x}{\cot(x)} = -\csc^2(x)\)
  19. \(\displaystyle \lzoo{x}{\sin^{-1}(x)} = \frac{1}{\sqrt{1-x^2}}\)
  20. \(\displaystyle \lzoo{x}{\cos^{-1}(x)} = -\frac{1}{\sqrt{1-x^2}}\)
  21. \(\displaystyle \lzoo{x}{\tan^{-1}(x)} = \frac{1}{1+x^2}\)
  22. \(\displaystyle \lzoo{x}{\csc^{-1}(x)} = -\frac{1}{\abs{x}\sqrt{x^2-1}}\)
  23. \(\displaystyle \lzoo{x}{\sec^{-1}(x)} = \frac{1}{\abs{x}\sqrt{x^2-1}}\)
  24. \(\displaystyle \lzoo{x}{\cot^{-1}(x)} = -\frac{1}{1+x^2}\)
