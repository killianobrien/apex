This description of \(D\) is not very easy to integrate. With polar, we can do better. Let \(R\) represent the unit disk, which can be described in polar simply as \(r\text{,}\) where \(0\leq r\leq 1\) and \(0\leq \theta\leq 2\pi\text{.}\) With \(x=r\cos \theta\) and \(y=r\sin\theta\text{,}\) the surface \(\surfaceS_2\) becomes
\begin{equation*} z=1-x^2-y^2 \Rightarrow 1-(r\cos\theta)^2-(r\sin\theta)^2 \Rightarrow 1-r^2\text{.} \end{equation*}
