In order to shift the graph to the right 3 units, we need to increase the \(x\)-value by 3 for every point. The straightforward way to accomplish this is simply to add 3 to the function defining \(x\text{:}\) \(x = t^2+t+3\text{.}\) To shift the graph down by 2 units, we wish to decrease each \(y\)-value by 2, so we subtract 2 from the function defining \(y\text{:}\) \(y = t^2-t-2\text{.}\) Thus our parametric equations for the shifted graph are \(x=t^2+t+3\text{,}\) \(y=t^2-t-2\text{.}\) This is graphed in FigureÂ 10.2.10.(a). Notice how the vertex is now at \((3,-2)\text{.}\)
