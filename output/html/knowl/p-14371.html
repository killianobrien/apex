In SectionÂ 13.3 we found that it was useful to differentiate functions of several variables with respect to one variable, while treating all the other variables as constants or coefficients. We can integrate functions of several variables in a similar way. For instance, if we are told that \(f_x(x,y) = 2xy\text{,}\) we can treat \(y\) as staying constant and integrate to obtain \(f(x,y)\text{:}\)
\begin{align*} f(x,y) \amp = \int f_x(x,y)\, dx\\ \amp = \int 2xy\, dx\\ \amp = x^2y + C\text{.} \end{align*}
