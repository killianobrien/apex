Solution 10.1.7.2.

The vertex is located halfway between the focus and directrix, so \((h,k) = (1,2.5)\text{.}\) This gives \(p=-0.5\text{.}\) Using Key Idea 10.1.6 we have the equation of the parabola as
\begin{equation*} y=\frac{1}{4(-0.5)}(x-1)^2+2.5 = -\frac12(x-1)^2+2.5\text{.} \end{equation*}
