Solution 13.5.3.16.1.
We find that
\begin{equation*} \frac{dz}{dt} = -\frac{2\cos^2(t) \sin(t) }{(1+\sin^2(t) )^2}-\frac{\sin(t) }{1+\sin^2(t) } \text{.} \end{equation*}
Setting this equal to \(0\text{,}\) finding a common denominator and factoring out \(\sin(t)\text{,}\) we get
\begin{equation*} \sin(t) \left(\frac{2\cos^2(t) +\sin^2(t) +1}{(1+\sin^2(t) )^2}\right)=0 \text{.} \end{equation*}
We have \(\sin(t) = 0\) when \(t = \pi n\text{,}\) where \(n\) is an integer. The expression in the parenthesis above is always positive, and hence never equal \(0\text{.}\) So all solutions are \(t=\pi n\text{,}\) n is an integer. In \([0,2\pi)\text{,}\) the only solutions are \(0\) and \(\pi\text{.}\)
