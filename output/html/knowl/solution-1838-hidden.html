We begin by finding parametrizations of \(C_1\) and \(C_2\text{.}\) As done in Example 15.3.12, parametrize \(C_1\) by creating the line that starts at \((1,0)\) and moves in the \(\la -1,1\ra\) direction: \(\vec r_1(t) = \la 1,0\ra + t\la -1,1\ra = \la 1-t, t\ra\text{,}\) for \(0\leq t\leq 1\text{.}\) We parametrize \(C_2\) with the familiar \(\vec r_2(t) = \la \cos t,\sin t\ra\) on \(0\leq t\leq \pi/2\text{.}\) For reference later, we give each function and its derivative below:
\begin{equation*} \vec r_1(t) = \la 1-t, t\ra, \vrp_1(t) = \la -1,1\ra\text{.} \end{equation*}
\begin{equation*} \vec r_2(t) = \la \cos t, \sin t\ra, \vrp_2(t) = \la -\sin t ,\cos t\ra\text{.} \end{equation*}
When \(\vec F = \vec F_1 = \la y, -x+1\ra\) (as shown in Figure 15.4.6.(a)), over \(C_1\) we have \(M = y =t\) and \(N = -x+1 = -(1-t)+1 = t\text{.}\) Using Definition 15.4.4, we compute the flux:
\begin{align*} \int_{C_1} \vec F\cdot \vec n\, ds \amp = \int_{C_1} \Big(M\,g'(t) - N\,\fp(t)\Big)\, dt\\ \amp = \int_0^1 \Big( t(1) - t(-1)\Big)\, dt\\ \amp = \int_0^1 2t\, dt\\ \amp = 1\text{.} \end{align*}
Over \(C_2\text{,}\) we have \(M = y = \sin t\) and \(N = -x+1 = 1-\cos t\text{.}\) Thus the flux across \(C_2\) is:
\begin{align*} \int_{C_1} \vec F\cdot \vec n\, ds \amp = \int_{C_1} \Big(M\,g'(t) - N\,\fp(t)\Big)\, dt\\ \amp = \int_0^{\pi/2}\Big((\sin t)(\cos t) - (1-\cos t)(-\sin t)\Big)\, dt\\ \amp = \int_0^{\pi/2} \sin t\, dt\\ \amp =1\text{.} \end{align*}
Notice how the flux was the same across both curves. This won't hold true when we change the vector field.
When \(\vec F = \vec F_2 = \la -x,2y-x\ra\) (as shown in Figure 15.4.6.(b)), over \(C_1\) we have \(M = -x = t-1\) and \(N = 2y-x = 2t-(1-t) = 3t-1\text{.}\) Computing the flux across \(C_1\text{:}\)
\begin{align*} \int_{C_1} \vec F\cdot \vec n\, ds \amp = \int_{C_1} \Big(M\,g'(t) - N\,\fp(t)\Big)\, dt\\ \amp = \int_0^1 \Big( (t-1)(1) - (3t-1)(-1)\Big)\, dt\\ \amp = \int_0^1 (4t-2)\, dt\\ \amp = 0\text{.} \end{align*}
Over \(C_2\text{,}\) we have \(M = -x = -\cos t\) and \(N = 2y-x = 2\sin t-\cos t\text{.}\) Thus the flux across \(C_2\) is:
\begin{align*} \int_{C_1} \vec F\cdot \vec n\, ds \amp = \int_{C_1} \Big(M\,g'(t) - N\,\fp(t)\Big)\, dt\\ \amp = \int_0^{\pi/2}\Big((-\cos t)(\cos t) - (2\sin t-\cos t)(-\sin t)\Big)\, dt\\ \amp = \int_0^{\pi/2} \big(2\sin^2 t-\sin t\cos t-\cos^2t\big)\, dt\\ \amp =\pi/4 - 1/2\approx 0.285\text{.} \end{align*}
