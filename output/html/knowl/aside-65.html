Note that we require \(h(x)\) to be a polynomial function in TheoremÂ 9.8.12. If we plug a function that is not polynomial into a power series, the result will no longer be a power series. If one is very careful about the centre and radius of convergence, it is technically possible to substitute the Taylor series for a general function \(h(x)\) into the Taylor series for \(f(x)\text{,}\) and the result will be the Taylor series for \(f(h(x))\text{.}\)
In practice, \(h(x)\) is typically a monomial function of the form \(h(x)=ax^n\text{.}\) For anything more complicated, rearranging the power series into a standard form becomes a nightmare.
