The integrand contains a composition of functions, leading us to think Substitution would be beneficial. Letting \(u=\ln(x)\text{,}\) we have \(du = 1/x\, dx\text{.}\) This seems problematic, as we do not have a \(1/x\) in the integrand. But consider:
\begin{equation*} du = \frac 1x\, dx \quad \Rightarrow \quad x\cdot du = dx\text{.} \end{equation*}
Since \(u = \ln(x)\text{,}\) we can use inverse functions and conclude that \(x = e^u\text{.}\) Therefore we have that
\begin{align*} dx \amp = x\cdot du\\ \amp = e^u\, du\text{.} \end{align*}
We can thus replace \(\ln(x)\) with \(u\) and \(dx\) with \(e^u\, du\text{.}\) Thus we rewrite our integral as
\begin{equation*} \int \cos(\ln(x))\, dx = \int e^u\cos u \, du\text{.} \end{equation*}
We evaluated this integral on the right in ExampleÂ 6.2.10. (This integral can also be found in a table of integrals). Using the result there, we have:
\begin{align*} \int \cos(\ln(x) )\, dx \amp = \int e^u\cos(u) \, du\\ \amp = \frac12e^u\big(\sin(u) + \cos(u) \big) + C\\ \amp = \frac12e^{\ln(x) } \big(\sin(\ln(x) ) + \cos(\ln(x) )\big)+C\\ \amp = \frac12x \big(\sin(\ln(x) ) + \cos(\ln(x) )\big)+C\text{.} \end{align*}
