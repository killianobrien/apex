With \(\divv \vec F = 1\text{,}\) we can integrate, recalling that \(dV = r\, dz\, dr\, d\theta\text{:}\)
\begin{equation*} \iiint_D\divv \vec F\, dV = \int_0^{2\pi}\int_0^1\int_0^{1-r^2} r\, dz\, dr\, d\theta = \frac{\pi}2\text{,} \end{equation*}
which matches our flux computation above.
