Along the curve \(y = x^4\text{,}\) \(\vec F\big(\vec r_2(t)\big) = \la x, x+y\ra = \la t, t+t^4\ra\text{.}\) We find \(\vrp_2(t) = \la 1, 4t^3\ra\text{.}\) The work performed along this path is
\begin{align*} \int_{C_2} \vec F\cdot d\vec r \amp = \int_0^1 \la t,t+t^4\ra\cdot\la 1,4t^3\ra\, dt\\ \amp = \int_0^1 \big(t + 4t^4+ 4t^7\big)\, dt\\ \amp = \big(\frac12t^2 + \frac45t^5 + \frac12t^8\big)\Big|_0^1 = 1.8 \text{ joules } \text{.} \end{align*}
