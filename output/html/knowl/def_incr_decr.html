Definition 3.3.2. Increasing and Decreasing Functions.

Let \(f\) be a function defined on an interval \(I\text{.}\)
  1. \(f\) is increasing on \(I\) if for every \(a\lt b\) in \(I\text{,}\) \(f(a) \lt f(b)\text{.}\)
  2. \(f\) is decreasing on \(I\) if for every \(a\lt b\) in \(I\text{,}\) \(f(a) \gt f(b)\text{.}\)
