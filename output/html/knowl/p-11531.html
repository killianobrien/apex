Note how this is the Triple Scalar Product, first seen in Theorem 11.4.6. Applying the identities given in the theorem shows that we can apply the Triple Scalar Product in any “order” we choose to find the volume. That is,
\begin{equation*} V = \abs{\vecu\cdot(\crossp vw)} = \abs{\vec u\cdot (\crossp wv)} = \abs{(\crossp uv)\cdot \vecw}, \text{ etc. } \end{equation*}
