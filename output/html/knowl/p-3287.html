If a function has the same concavity on adjacent intervals \((a,b)\) and \((b,c)\text{,}\) and the function is continuous at \(b\text{,}\) we should combine the intervals, and state the result as \((a,c)\text{.}\) However, if \(b\) is a point of discontinuity, we must omit it from our intervals.
