For a numerical approximation, make a table:
\(x\) \(e^[-e^(1/x)]\)
\(-0.1\) \(0.999955\)
\(-0.01\) \(1\)
\(-0.001\) \(1\)
\(0.001\) \(0\)
\(0.01\) \(0\)
\(0.1\) \(0\)
For a graphical approximation:
It appears that when \(x\) is close to \(0\text{,}\) that \(e^{-e^{1/x}}\) approaches different values from the left and right. So \(\lim_{x\to 0}e^{-e^{1/x}}\) does not exist.
