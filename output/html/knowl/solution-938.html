Solution 9.8.16.1.
Given a value \(x\text{,}\) the magnitude of the error term \(R_n(x)\) is bounded by
\begin{equation*} \abs{R_n(x)} \leq \frac{\max\abs{\,f^{(n+1)}(z)}}{(n+1)!}\abs{x^{(n+1)}}\text{,} \end{equation*}
where \(z\) is between \(0\) and \(x\text{.}\)
Note that \(\abs{f^{(n+1)}(x)} = \frac{(n+1)!}{(1-x)^{n+2}}\text{.}\)
If \(-1\lt x\lt 0\text{,}\) then \(x\lt z\lt 0\) and \(f^{(n+1)}(z) =\frac{(n+1)!}{(1-z)^{n+2}}\lt \frac{(n+1)!}{(1-x)^{n+2}}\text{.}\) Thus
\begin{equation*} \abs{R_n(x)} \leq \frac{(n+1)!}{(1-x)^{n+2}}\frac{1}{(n+1)!}\abs{x^{n+1}}= \frac{(x-1)^{n+1}}{n+1}\text{.} \end{equation*}
For a fixed \(x\text{,}\)
\begin{equation*} \lim_{n\to\infty} \frac{(x-1)^{n+1}}{n+1}=0 \end{equation*}
since \(\abs{x}\lt 1\text{,}\) hence
\begin{equation*} \frac{1}{1-x} = \infser[0] x^n \text{ on } (-1,0)\text{.} \end{equation*}
