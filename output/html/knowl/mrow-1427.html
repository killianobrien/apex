\begin{align*} \inflim \left(a_n \right)^{1/n} \amp =\inflim \left(\frac{n^4}{(\ln(n))^n}\right)^{1/n}\\ \amp = \inflim \frac{\big(n^{4/n}\big)}{\ln(n)}\\ \end{align*}

The limit of the numerator must be found using L'Hospital's Rule for indeterminate powers

\begin{align*} \inflim \left(n^{4/n}\right) \amp = \inflim e^{\ln\left(n^{4/n}\right)}\\ \amp = \inflim e^{{4\ln\left(n\right)}/n}\\ \end{align*}

Now apply L'Hospital's to the expression in the exponent:

\begin{align*} \amp \stackrel{\, \text{ by LHR } \, }{=} \inflim e^{4/n}\\ \amp = e^0=1 \end{align*}
