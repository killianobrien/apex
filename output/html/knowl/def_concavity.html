Definition 3.4.2. Concave Up and Concave Down.

Let \(f\) be continuous on an interval \(I\text{.}\) The graph of \(f\) is concave up on \(I\) if for any \(a\lt b\) in \(I\text{,}\)
\begin{equation} f\left(\frac{a+b}{2}\right) \lt \frac{f(a)+f(b)}{2}\text{.}\tag{3.4.1} \end{equation}
The graph of \(f\) is concave down on \(I\) if for any \(a\lt b\) in \(I\text{,}\)
\begin{equation} f\left(\frac{a+b}{2}\right) \gt \frac{f(a)+f(b)}{2}\text{.}\tag{3.4.2} \end{equation}
