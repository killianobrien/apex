Integrating \(x+2\) is simple. The fraction can be integrated by setting \(u = x^2+2x+1\text{,}\) giving \(du = (2x+2)\, dx\text{.}\) This is very similar to the numerator. Note that \(du/2 = (x+1)\, dx\) and then consider the following:
\begin{align*} \int \frac{x^3+4x^2+8x+5}{x^2+2x+1}\, dx \amp = \int \left(x+2 + \frac{3x+3}{x^2+2x+1}\right)\, dx\\ \amp = \int (x+2)\, dx + \int \frac{3(x+1)}{x^2+2x+1}\, dx\\ \amp = \frac12x^2+2x+C_1 + \int \frac{3}{u}\frac{du}{2}\\ \amp = \frac12x^2+2x+C_1 + \frac32\ln\abs{u} + C_2\\ \amp = \frac12x^2+2x+\frac32\ln\abs{x^2+2x+1} + C\text{.} \end{align*}
