Let \(f\) be a function defined on \((a,\infty)\) for some number \(a\text{.}\) The limit of \(f\) at infinity is \(L\text{,}\) denoted \(\lim_{x\to\infty} f(x)=L\text{,}\) if for every \(\epsilon \gt 0\) there exists \(M \gt a\) such that if \(x \gt M\text{,}\) then \(\abs{f(x)-L}\lt \epsilon\text{.}\)
