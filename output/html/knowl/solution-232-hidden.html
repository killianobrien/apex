A closed interval is not given, so we find the extreme values of \(f\) on its domain. \(f\) is defined whenever \(1-x^2\geq 0\text{;}\) thus the domain of \(f\) is \([-1,1]\text{.}\) Evaluating \(f\) at either endpoint returns 0.
Using the The Chain Rule, we find \(\fp(x) = -x\big/\sqrt{1-x^2}\text{.}\) The critical points of \(f\) are found when \(\fp(x) = 0\) or when \(\fp\) is undefined. It is straightforward to find that \(\fp(x) = 0\) when \(x=0\text{,}\) and \(\fp\) is undefined when \(x=\pm 1\text{,}\) the endpoints of the interval (which are in the domain of \(f\text{.}\)) The table of important values is given in Figure 3.1.30. The maximum value is \(1\text{,}\) and the minimum value is \(0\text{.}\)
