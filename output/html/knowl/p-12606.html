We can find \(a_\text{T}\) and \(a_\text{N}\) directly with dot products:
\begin{align*} a_\text{T} \amp = \vat\cdot \unittangent(t) = \frac95\cos(t) \sin(t) -\frac95\cos(t) \sin(t) +0 = 0.\\ a_\text{N} \amp = \vat\cdot \unitnormal(t) = 3\cos^2(t) +3\sin^2(t) + 0 = 3\text{.} \end{align*}
