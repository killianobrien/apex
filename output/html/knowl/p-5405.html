Since we set \(u = x+3\text{,}\) we can also state that \(u-3 = x\text{.}\) Thus we can replace \(x\) in the integrand with \(u-3\text{.}\) It will also be helpful to rewrite \(\sqrt{u}\) as \(u^\frac12\text{.}\)
\begin{align*} \int x\sqrt{x+3} \, dx \amp = \int (u-3)u^\frac12\, du\\ \amp = \int \big(u^\frac32 - 3u^\frac12\big) \, du\\ \amp = \frac25u^\frac52 - 2u^\frac32 + C\\ \amp = \frac25(x+3)^\frac52 - 2(x+3)^\frac32 + C\text{.} \end{align*}
