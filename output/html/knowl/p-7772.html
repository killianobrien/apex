This statement says that if we know the solution (\(y\)-value) to the initial value problem for some given \(x\)-value, we can find an approximation for the solution at the value \(x+h\) by taking our \(y\)-value and adding \(h\) times the function \(f\) evaluated at the \(x\) and \(y\) values. Euler's method uses the initial condition of an initial value problem as the starting point, and then uses the above idea to find approximate values for the solution \(y\) at later \(x\)-values.
