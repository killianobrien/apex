Theorem 14.7.7. Triple Integration in Cylindrical Coordinates.

Let \(w=h(r,\theta,z)\) be a continuous function on a closed, bounded region \(D\) in space, bounded in cylindrical coordinates by \(\alpha \leq \theta \leq \beta\text{,}\) \(g_1(\theta)\leq r \leq g_2(\theta)\) and \(f_1(r,\theta) \leq z \leq f_2(r,\theta)\text{.}\) Then
\begin{equation*} \iiint_D h(r,\theta,z)\, dV = \int_\alpha^\beta\int_{g_1(\theta)}^{g_2(\theta)}\int_{f_1(r,\theta)}^{f_2(r,\theta)}h(r,\theta,z) r\,dz\,dr\,d\theta\text{.} \end{equation*}
