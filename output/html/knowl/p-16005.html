Recall that function is one to one on its domain if the function never repeats an output value over the domain. In the case of \(\vec r(u,v)\text{,}\) \(\vec r\) is one to one if \(\vec r(u_1,v_1) \neq \vec r(u_2,v_2)\) for all points \((u_1,v_1) \neq (u_2,v_2)\) in the domain of \(\vec r\text{.}\)
