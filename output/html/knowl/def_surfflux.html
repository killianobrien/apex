Definition 15.6.6. Flux over a surface.

Let \(\vec F\) be a vector field with continuous components defined on an orientable surface \(\surfaceS\) with normal vector \(\vec n\text{.}\) The flux of \(\vec F\) across \(\surfaceS\) is
\begin{equation*} \text{ Flux } = \iint_\surfaceS \vec F\cdot \vec n\, dS\text{.} \end{equation*}
If \(\surfaceS\) is parametrized by \(\vec r(u,v)\text{,}\) which is smooth on its domain \(R\text{,}\) then
\begin{equation*} \text{ Flux } = \iint_R \vec F\big(\vec r(u,v)\big)\cdot (\vec r_u\times \vec r_v)\, dA\text{.} \end{equation*}
