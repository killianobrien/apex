Theorem 1.3.7. Limits of Common Functions.

Let \(c\) be a real number in the domain of the given function and let \(n\) be a positive integer. The following limits hold:
  1. \(\displaystyle \lim\limits_{x\to c}\sin(x) = \sin(c)\)
  2. \(\displaystyle \lim_{x\to c}\cos(x) = \cos(c)\)
  3. \(\displaystyle \lim_{x\to c}\tan(x)= \tan(c)\)
  4. \(\displaystyle \lim_{x\to c}\csc(x) = \csc(c)\)
  5. \(\displaystyle \lim_{x\to c}\sec(x) = \sec(c)\)
  6. \(\displaystyle \lim_{x\to c}\cot(x) = \cot(c)\)
  7. \(\lim_{x\to c}a^x = a^c\text{,}\) if \(a \gt 0\)
  8. \(\displaystyle \lim_{x\to c}\ln(x) = \ln(c)\)
  9. \(\displaystyle \lim_{x\to c}\sqrt[n]{x} = \sqrt[n]{c}\)
