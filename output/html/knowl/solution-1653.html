Solution 14.2.9.1.

We wish to evaluate \(\iint_R \big(xy+e^y\big)\, dA\text{.}\) As \(R\) is a rectangle, the bounds are easily described as \(3\leq x\leq 4\) and \(1\leq y\leq 2\text{.}\)
Using the order \(dy\, dx\text{:}\)
\begin{align*} \iint_R\big(xy+e^y\big) \, dA \amp = \int_3^4\int_1^2\big(xy+e^y\big)\, dy \, dx\\ \amp = \int_3^4 \left(\left.\left[\frac12xy^2+e^y\right]\right|_1^2\, \right) dx\\ \amp = \int_3^4\left(\frac 32x + e^2-e\right)dx\\ \amp = \left.\left(\frac 34x^2 + \big(e^2-e\big)x\right)\right|_3^4\\ \amp = \frac {21}4+ e^2-e\approx 9.92\text{.} \end{align*}
Now we check the validity of Fubini's Theorem by using the order \(dx\, dy\text{:}\)
\begin{align*} \iint_R\big(xy+e^y\big) \, dA \amp = \int_1^2\int_3^4\big(xy+e^y\big)\, dx \, dy\\ \amp = \int_1^2\left(\left.\left[\frac12x^2y+xe^y\right]\right|_3^4\right)dy\\ \amp = \int_1^2\left(\frac72y+e^y\right)\, dy\\ \amp = \left.\left(\frac74y^2+e^y\right)\right|_1^2\\ \amp =\frac{21}4+e^2-e\approx 9.92\text{.} \end{align*}
Both orders of integration return the same result, as expected.
